Create New Program

Use this flow to register a new cohort, assign coordinators, and auto-generate access codes. In the production app this will POST to the program creation endpoint.

Steps After Creation

  1. Share the auto-generated program code with mentors and students.
  2. Import existing resources or copy from a previous cohort.
  3. Invite mentors and assign them to student groups.
  4. Publish welcome announcements through the communications hub.
